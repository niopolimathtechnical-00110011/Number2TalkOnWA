## 🗣️ Português

### Sobre o Projeto

**Number2Talk on WhatsApp** é um utilitário Android ultra-leve (~54KB) que permite abrir conversas no WhatsApp diretamente de qualquer número de telefone copiado na área de transferência ou digitado manualmente.

Com um **balão flutuante** que aparece automaticamente ao copiar um número, ou através do **campo de digitação manual**, basta um toque para iniciar a conversa - sem precisar salvar o contato!

### ✨ Funcionalidades

| Funcionalidade | Descrição |
|----------------|-------------|
| 🎈 **Balão Flutuante** | Aparece automaticamente ao copiar um número de telefone |
| 📝 **Digitação Manual** | Campo para digitar o número diretamente |
| 📋 **Colar Rápido** | Botão para processar números do clipboard |
| 📜 **Histórico de Conversas** | Registra todas as conversas iniciadas |
| 🐞 **Reporte de Erros** | Envia logs e descrição do erro via WhatsApp |
| ☕ **Doações** | Informações para apoiar o projeto via PIX |
| 📱 **Leve e Rápido** | Apenas ~54KB, não consome bateria |
| 🔄 **3 Abas Organizadas** | Conversa Rápida, Sobre o App e Ferramentas |
| 📊 **Relatório Automático** | Envia logs detalhados para o desenvolvedor |
| 🔢 **Formatação Automática** | Ajusta números brasileiros automaticamente |

### 📋 Como Usar

#### Método 1: Balão Flutuante (Recomendado)
1. **Baixe o APK** na seção [Releases](https://github.com/niopolimathtechnical-00110011/Number2TalkOnWA/releases)
2. **Instale** no seu dispositivo Android (5.0+)
3. **Abra o app** e conceda a permissão de "Sobrepor outros apps"
4. **Copie qualquer número** de telefone (ex: `+55 66 9 8432 8877`)
5. **Toque no balão flutuante** que aparecerá na tela
6. **O WhatsApp abrirá** diretamente na conversa com o número

#### Método 2: Digitação Manual
1. Abra o app e vá para a aba **"📱 Conversa Rápida"**
2. Digite o número no campo telefone (aceita 10, 11 ou 13 dígitos)
3. Opcionalmente, adicione um nome para o contato
4. Clique em **"💬 Conversar"**
5. O WhatsApp abrirá diretamente na conversa

### 🛠️ Ferramentas

O app possui uma aba **"🛠️ Ferramentas"** com:
- 📜 **Histórico de Conversas** - Visualize e copie conversas anteriores
- 🔍 **Verificar Permissão de Sobreposição** - Garanta que o balão funcione
- 📋 **Testar Leitura do Clipboard** - Verifique o que está copiado
- 🐞 **Reportar Erro via WhatsApp** - Envie descrição + logs para o desenvolvedor

## 🇬🇧 English

### About

**Number2Talk on WhatsApp** is an ultra-lightweight (~54KB) Android utility that opens WhatsApp conversations directly from any phone number copied to the clipboard or typed manually.

With a **floating bubble** that automatically appears when copying a number, or through the **manual input field**, just one tap starts the conversation - no need to save the contact!

### Features

| Feature | Description |
|---------|-------------|
| 🎈 **Floating Bubble** | Appears automatically when copying a phone number |
| 📝 **Manual Input** | Field to type the number directly |
| 📋 **Quick Paste** | Button to process numbers from clipboard |
| 📜 **Conversation History** | Records all initiated conversations |
| 🐞 **Error Reporting** | Sends logs and error description via WhatsApp |
| ☕ **Donations** | Support the project via PIX |
| 📱 **Lightweight & Fast** | Only ~54KB, no battery drain |
| 🔄 **3 Organized Tabs** | Quick Chat, About, Tools |

### How to Use

#### Method 1: Floating Bubble
1. **Download the APK** from [Releases](https://github.com/niopolimathtechnical-00110011/Number2TalkOnWA/releases)
2. **Install** on your Android device (5.0+)
3. **Open the app** and grant "Draw over other apps" permission
4. **Copy any phone number** (e.g., `+55 66 9 8432 8877`)
5. **Tap the floating bubble** that appears on screen
6. **WhatsApp opens** directly to that number

#### Method 2: Manual Typing
1. Open the app and go to **"📱 Quick Chat"** tab
2. Type the number (accepts 10, 11 or 13 digits)
3. Optionally add a name for the contact
4. Click **"💬 Chat"**
5. WhatsApp opens directly to the conversation
